# 🤖 H-AIBuddy
### *Your Intelligent AI Guardian*

H-AIBuddy is a robotic AI companion designed to provide smart assistance and protection.

## 🌟 Vision
To build a helpful AI that manages data securely and assists users through a friendly interface.

## 📁 Project Map
- `src/`: Core AI logic and database connections.
- `data/`: Secure storage for the AI's memory.
- `docs/`: Roadmap and project planning.

## 🚀 Get Started
Run `python src/db/repository.py` to wake up the AI's memory!
