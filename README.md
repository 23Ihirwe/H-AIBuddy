## 🌟 The Vision
**H-AIBuddy** is more than just code; it is a "Leadership Core" designed to assist mothers in managing their family's educational journey. The project focuses on providing a safe, intelligent, and adaptive environment where technology meets the heart of a home.

Our mission is to bridge the gap between AI and human care by providing a guardian that teaches, remembers, and grows with your family.

---

## 🛠️ Key Features

### 👩‍🏫 Adaptive Mother-Child Interview
H-AIBuddy starts with a respectful, permission-based interview. It adapts its logic based on:
*   **The Mother's Profile**: Career status, education level, and personal learning goals.
*   **The Child's Age**: From infants (2 months) to adults (18+), providing age-appropriate advice and "Mastery Tracks."

### 📚 The Multi-Disciplinary Learning Hub
A professional visual dashboard providing notes and slides for:
*   **🌍 Language Mastery**: Specialized modules for **English, Deutsch, French, and Creole**.
*   **🌌 Science & Physics**: Deep dives into the laws of motion and the natural world.
*   **🧮 Logic & Math**: From basic numbers for toddlers to advanced logic for adults.
*   **💼 Entrepreneurship**: Training the next generation of business leaders and strategists.

### 🧠 AI Memory & Time Analytics
*   **Session Tracking**: Records exactly how much time is spent on each lesson.
*   **Adaptive Memory**: The robot listens to the mother's thoughts and saves them to a downloadable session report.
*   **Visual Slides**: Clean, professional data-analytics-style slides for every subject.

## 🚀 Getting Started

### 1. Installation
To run this project locally, clone the repository and install the requirements:
```bash
git clone https://github.com
cd H-AIBuddy
pip install -r requirements.txt
```

### 2. Run Locally
Launch the interactive visual interface:
```bash
streamlit run app.py
```

## 🛡️ Security & Privacy
H-AIBuddy is designed with privacy in mind. User data is stored locally on the user's PC, and the `.gitignore` ensures that personal databases are never leaked to the public internet.
